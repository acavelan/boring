Boring
=======

Are you bored ? In the middle of a meeting or a class ? Monitor it.

Usage
======

Bash:
- Date: hh:mm:ss
- usage example: ./boring.sh 10:15:00 12:15:15

What it looks like
===================

02:12:42  
Boring time remaining: 4638s 14%
